## 3. Idea 快捷键

#### 3.1 编辑

1. Ctrl + Shift + Space 示例（智能补全）
2. Ctrl + P 示例（方法参数列表）
3. Ctrl + Q 示例（API 说明文档）
4. Alt + Insert 示例（自动生成与类相关的代码）
5. Ctrl + O 示例（该类中所有覆盖或者实现的方法列表）
6. Ctrl + Alt + T 示例（生成具有环绕性质的代码）
7. Ctrl + Shift + V 示例（剪切或拷贝的代码历史记录中，选择粘贴的内容）



#### 3.2 查找和替换

1. Ctrl + F （查找）
2. Ctrl + Shift + F （通过路径查找）
